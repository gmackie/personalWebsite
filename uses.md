## Stack

**Languages:** Ruby (Jekyll), Python, JavaScript/TypeScript, Nix, Bash

**Infrastructure:** Nix, ForgeGraph, Caddy, Podman, Kubernetes

**Hosting:** Hetzner (production), NUC homelab (staging), Tailscale (mesh)

**Development:** Claude Code, JetBrains IDEs, JetBrains Mono in the terminal

**Source control:** Gitea (self-hosted), GitHub (mirrors)

**CI/CD:** Gitea Actions, Nix builds, ForgeGraph deploys

**This site:** Jekyll, SCSS, served as a static site via ForgeGraph edge

## Hardware

MacBook Pro (daily driver), Intel NUC (homelab/staging), Hetzner dedicated (production)

## Design

Satoshi (display), Instrument Sans (body), JetBrains Mono (code/data). Warm editorial palette with burnt sienna accent. Full spec in [DESIGN.md](https://github.com/gmackie/personalWebsite/blob/main/DESIGN.md).
